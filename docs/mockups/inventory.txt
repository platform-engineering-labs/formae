================================================================================
DATA FIELDS COVERAGE
================================================================================

Resource fields shown:
  Table:  NativeID, Stack, Type, Label
  Detail: + Target, Managed, Properties (formatted tree),
          ReadOnlyProperties (if useful), Group, Schema

Target fields shown:
  Table:  Label, Namespace, Discoverable, Config (compact, filtered)
  Detail: + full Config JSON, Version

Stack fields shown:
  Table:  Label, Description, Policies (formatted summary)
  Detail: + CreatedAt, full policy configs

PolicyInventoryItem fields shown:
  Table:  Label, Type, Config (compact), AttachedStacks
  Detail: + full Config JSON, all attached stacks (untruncated)
