    ✓ Updated aws        2.1.0 → 2.1.1

  Notes:
  - Spinner line while a download is in flight, replaced by the ✓ line
    (simple line rewrite, not a bubbletea app)

    (improvement over today's version-less "Updated <name>")

  - Agent-restart hint keeps the ! marker (Warning style)
  - Uninstall mirrors install with "Removed <name>"
  - Failures render as error panels (see error-panels.txt: Plugin Not
    Found, Plugin Dependency Conflict) after the per-plugin lines
