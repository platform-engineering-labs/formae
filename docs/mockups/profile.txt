================================================================================
CLASSIFICATION
================================================================================

  profile list / current               improved formatting (print-and-exit)
  profile use / create / save / delete improved formatting (✓ ack lines)
  profile edit                         unchanged (execs $EDITOR)
  profile diff                         unchanged (execs diff -u, exit codes
                                       passed through; colorizing diff
                                       output is the pager's job, not ours)

  The global --profile flag has no rendering surface of its own; the
  active profile name appears in the status/watch header when it is not
  the default (see status-watch.txt header bar).
